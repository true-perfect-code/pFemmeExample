## 📌 Projektbeschreibung

**pFemme** ist eine DSGVO-konforme Webanwendung für Patientinnen von Frauenarztpraxen. Sie ermöglicht die vertrauliche Erfassung und Auswertung von Menstruationszyklen – mit dem Ziel, die Patientinnenbindung durch eine moderne, sichere und datensouveräne Gesundheits-App zu stärken.

Dieses Repository enthält einen **bereinigten Code-Auszug** unseres produktiven Systems und dient als **Showcase** für unsere Architektur- und Entwicklungsstandards. Es zeigt, wie wir komplexe, mehrsprachige und barrierefreie .NET-Lösungen mit modernem Stack umsetzen.

> **Hinweis:** Dies ist ein öffentliches Beispiel-Repository. Die vollständige Produktiv-Umgebung läuft unter [https://pfemme.de/](https://pfemme.de/).

## 🧩 Technologie-Stack

| Bereich | Technologie |
|---------|-------------|
| **Frontend/Backend** | Blazor Server (.NET 10) |
| **Datenbank** | Microsoft SQL Server (Azure SQL / Managed SQL) |
| **Authentifizierung** | Individuell (E-Mail/Passwort) + Google, Microsoft, Apple |
| **KI & Analysen** | Azure OpenAI (RAG), Azure AI Search |
| **Hosting** | Azure App Services / IONOS / beliebig |
| **Barrierefreiheit** | WCAG 2.1 AA, dynamischer Sprachwechsel, Schriftgrößenanpassung |
| **Testing** | xUnit, bUnit |
| **CI/CD** | GitHub Actions |
